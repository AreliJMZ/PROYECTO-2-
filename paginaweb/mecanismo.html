MECANISMOS Y ELEMENTOS DE JAVASCRIPT

Lenguaje orientado a objetos

Dado que HTML es un lenguaje de maracas, existe una etiqueta (apertura y cierre) especial para señalar que lo que aparece a continuación debe ser analizado y ejecutado por el intérprete de JavaScrip en lugar de por el intérprete de HTML.










Elementos de un programa de JavaScript

Un programa es un conjunto ordenado de instrucciones diseñado para llevar a cabo una tarea concreta. Donde los programas están formados por instrucciones que determinan las tareas que se realizan y el orden en que estas se llevan a cabo.
