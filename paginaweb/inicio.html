Misión del proyecto

JavaScript es un lenguaje de programación ampliamente utilizado en el desarrollo web. Fue creado originalmente para hacer que las páginas web fueran interactivas y dinámicas al permitir la manipulación del contenido de una página y la respuesta a eventos del usuario sin necesidad de recargar la página completa.



Creado el 4 de diciembre de 1995


El primer motor de JavaScript fue creado por Brendan Eich en Netscape Communications Corporation, para el navegador web Netscape Navigator.
