Aprendamos a programar con JavaScript

JavaScript cuentan con:


Constantes:

Representan entidades que no se modifican durante la ejecución del programa. Pueden tener un identificador o carecer del mismo.







Variables:

Posiciones de memoria a las que se asigna un nombre y a las que se puede acceder para obtener el valor que almacenan, y las que se pueden asignar son como: Número, Cadena, Booleano, Array, Objeto, Función, Símbolo, Undefinido, Nul, BigInt.




Operadores:

Símbolos que representan una operación de naturaleza matemática, relacional o lógica.








Instrucciones o sentencias:

Permiten organizar el flujo de ejecución de un programa estableciendo el orden en el que se encadenan las acciones del mismo.





Funciones :

Agrupaciones de instrucciones diseñadas para resolver un problema concreto
